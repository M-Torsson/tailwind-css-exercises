# Tailwind CSS Practice Exercises

This repository contains my solutions for the Tailwind CSS practice exercises.

The purpose of this work is to practice using Tailwind CSS utility classes to build different UI components and layouts. The exercises cover typography, spacing, colors, cards, buttons, responsive layouts, grids, navigation, flexbox alignment, sticky sidebar layout, galleries, and responsive ordering.

## Completed Exercises

I completed the main exercises from 1 to 15:

1. Typography and Spacing
2. Colorful Alert Boxes
3. Simple Card with Shadow
4. Circular Profile Badge
5. Interactive Buttons
6. Responsive Two-Column Layout
7. Product Grid
8. Hero Section with Overlay
9. Pricing Table Comparison
10. Simple Navigation Bar
11. Custom Flexbox Alignment
12. Feature List with Icons
13. Sticky Sidebar Layout
14. Gallery with Aspect Ratios
15. Responsive Order Change
